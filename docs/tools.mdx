## bash-helpers

Multi-bash (ubu, msys, gitbash, wsl) helpers to easy install software/python/vscode packages, setup os (dark mode, clean taskbar/clutter/unused), sync dotfiles, manage git repos, and more.  More info at [project page](https://github.com/alanlivio/bash-helpers) <FontAwesomeIcon icon={faGithub}/>
